# llamafile-tools

My tools and scripts for generating & using `Llamafile`s.

* `gen_llamafile.sh`: generates a llamafile with given HuggingFace model

* helper scripts
  * `answer.sh`: answers given text
  * `summarize.sh`: summarizes given text
